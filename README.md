# 🏦 Bank Management System (Java OOP Project)

## 📌 Overview

This is a **console-based Bank Management System** built using Java.
It demonstrates core **Object-Oriented Programming (OOP)** concepts like encapsulation, object interaction, and basic real-world logic.

---

## 🚀 Features

* Create bank account
* Deposit money
* Withdraw money
* Check account balance
* Display all customers

---

## 🧠 OOP Concepts Used

* **Encapsulation** → Private variables with getters
* **Abstraction** → Service layer (`BankService`)
* **Object Composition** → Customer has BankAccount
* **Modular Design** → Separate classes for each responsibility

---

## 📁 Project Structure

```
bank-management/
 ├── BankAccount.java
 ├── Customer.java
 ├── BankService.java
 └── Main.java
```

---

## ⚙️ How to Run

### 🔹 Using Terminal

```
javac *.java
java Main
```

### 🔹 Using IDE (Eclipse / IntelliJ)

1. Import project
2. Open `Main.java`
3. Click **Run ▶️**
